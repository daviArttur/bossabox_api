## Install packages ️🔧

```
$ npm install
```

## Running API 🐳

```
$ docker compose up -d
```

## Push db schema 💿

```
$ npx prisma db push
```

## Get admin access token🛡️

```
$ npm run get:token
```

## Documentation 📖

```
http://localhost:3000/api/swagger

```

## Testing 🔥

```
# unit
$ npm run test

# integration
$ npm run test:int // docker is required
```
